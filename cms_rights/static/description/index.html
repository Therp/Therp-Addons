Features
  • Website access by two groups: i)Webshop Manager(WM) ii)Marketing Manager(MM)
  • MMs are allowed to save and propose changes to pages they are associated with
  • Restrict MMs to use some editor functionality of website
  • Both groups can save pages as templates and reuse them later
  • Page creation request by MM will send to WM
  • Proposed changes stay unpublished until approved by WM
  • WMs can revert main page template by discarding all content changes after MM requests
  • Both groups will get notification by email on proposal and approval respectively
How it Works

Proposal & MM access rights

> MMs have limited access to font-properties(except font-size, font-color) and snippet functionalities(except resize,

   background,color)

> MMs are assigned with some limited set of snippets(except Banner,Text-Image)

> MMs can use pictures that have already been uploaded to the system

> MMs can save their changes as draft and reuse them for proposal

> MM can create page request & content changes on newly created page will not be proposed until approval of page request

> MMs can preview their own change requests from 'Website Admin' menu

Approval & WM access rights

> WMs have full access rights to access all pages available in the website

> WMs are able to approve or reject a proposal of MM

> WMs can preview changes before approving them

> WMs can re-set main page template by removing all content changes after MM requests

> WM can redirect page url from 'Website Admin->SEO Redirections'

Storage of webpage templates

Save: Both groups can save a page as a template from 'content->Save as Template'

Reuse: New page can be created from 'content->New Page Templates' by selecting saved page templates

WM can use all saved templates at time of new page creation but MM can use only their own

E-mail notification

WMS and MMs will get notify by email based on proposal and approval respectively
